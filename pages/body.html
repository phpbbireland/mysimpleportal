This page was produced using simple html code!


Our aim with MySimplePortal is to produce a package that allows users to build and maintain a web site with little or no programming skills.

If you can write a little html (or use a programs that does it for you), you can build a site with MySimplePotal...

This demonstration contains a few sample pages but you can add as many as you like...
We have also include a few of the standard blocks/widgets to show what can be done in a few minutes...
